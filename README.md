# KasperGet



KasperGet — это легковесная и современная утилита для удаленного мониторинга состояния смартфона прямо с экрана персонального компьютера под управлением Windows. Проект состоит из двух частей: фонового сервиса для Android (написанного на Java/Kotlin) и стильного графического дашборда для Windows (написанного на Python с использованием CustomTkinter).

## Основные возможности



### Windows Клиент (Дашборд):

* **Автоматическое обновление:** Опрашивает смартфон каждые 5 секунд без необходимости ручного клика.

* **Экран настроек:** Удобное меню с переключателями для гибкой настройки алертов.

* **Голосовые оповещения (TTS):** Компьютер может озвучить на русском языке критические события (низкий заряд, полная зарядка или любое изменение параметров).

* **Всплывающие уведомления Windows (Toast):** Нативные системные пуши в углу экрана, которые дублируют статус устройства.

* **Контроль связи:** Если телефон отключится от сети, клиент мгновенно сбросит датчики в состояние "нет связи", подсветит статус красным и предупредит вас алертом.

* **Стабильный UI:** Интерфейс выполнен в стиле Windows 11 Fluent Design без использования сторонних графических символов, что гарантирует 100% корректное отображение на любой системе.



### Android Сервис:

* **Полноценный Foreground Service:** Работает в фоне и не закрывается агрессивными механизмами энергосбережения современных оболочек (проверено на Realme/Oppo с Android 14).

* **Сбор подробных метрик:**

    * Текущий уровень заряда и температура батареи.

    * Точный уровень яркости экрана.

    * Аптайм (время работы устройства с момента включения).

    * Название Wi-Fi сети (SSID) и точный уровень сигнала в dBm.

    * Свободное и общее место во внутренней памяти, а также объем оперативной памяти (RAM).

    * Имя открытого в данный момент приложения на экране смартфона.

## Технологический стек



* **Android:** Java/Kotlin, Android SDK (Foreground Service, BatteryManager, WifiManager, UsageStatsManager).

* **Windows:** Python 3.x, customtkinter (графический интерфейс), requests (сетевые запросы), встроенный PowerShell (для генерации системных уведомлений и синтеза речи Windows без тяжелых сторонних библиотек).

## Инструкция по установке и запуску



Вам не нужно ничего компилировать или настраивать среды разработки. Все необходимые исполняемые файлы уже собраны и готовы к работе.



### 1. Настройка смартфона (Android)

1. Возьмите из архива готовый файл `KasperGet.apk` и установите его на свой смартфон.

2. При первом запуске предоставьте приложению все запрашиваемые разрешения (доступ к геолокации для получения параметров Wi-Fi и доступ к статистике использования для определения активного приложения).

3. Нажмите кнопку запуска сервиса на главном экране. В шторке уведомлений появится закрепленный плеер активной службы.

4. Запомните IP-адрес, который отобразится в приложении (например, `192.168.1.7`).



### 2. Запуск на компьютере (Windows)

1. Распакуйте папку с Windows-клиентом.

2. Откройте файл конфигурации (или исходный скрипт) и впишите туда IP-адрес вашего телефона:

   

   self.PHONE_IP = "192.168.x.x"  # Укажите IP вашего смартфона







3. Запустите готовый исполняемый файл `KasperGet.exe` (или скрипт `KasperGet.pyw`, если используете исходный код).

## Сборка из исходного кода под Windows (.EXE)



Если вы внесли изменения в Python-код и хотите пересобрать проект в один независимый исполняемый файл, используйте утилиту `PyInstaller`.



1. Установите упаковщик через PowerShell:



python -m pip install pyinstaller customtkinter requests









2. Выполните команду сборки в папке проекта (выберите подходящий вариант):

**Вариант А:**



pyinstaller --noconsole --onefile --collect-all customtkinter main.py









**Вариант Б (Если команда не распознается глобально):**


& "C:\Users\admin\AppData\Local\Python\pythoncore-3.14-64\Scripts\pyinstaller.exe" --noconsole --onefile --collect-all customtkinter main.py









3. Заберите готовый файл из появившейся папки `dist/`.

## Важные примечания (Траблшутинг)



* **Одна сеть:** Компьютер и смартфон обязательно должны быть подключены к одному роутеру (к одной локальной Wi-Fi сети), иначе клиент не сможет достучаться до телефона.

* **Android 14+:** В приложении учтены все современные требования безопасности Android, что гарантирует его стабильную работу в фоне даже при жестких политиках энергосбережения.
