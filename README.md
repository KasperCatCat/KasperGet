# KasperGet
KasperGet — легкая система мониторинга смартфона Realme с ПК под Windows, созданная на чистом вайбкодинге.  Фоновый Android-сервис собирает метрики (заряд, температура, Wi-Fi, RAM), а красивый дашборд на Python (CustomTkinter) обновляет их каждые 5 сек, озвучивает голосом и шлет пуши при полной зарядке или потере связи. Всё в один .EXE и .APK!
